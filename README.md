
# ConfrontSpend Expense Tracker

A simple HTML-based expense tracker that focuses on tracking **expenses** only. It features a modern, customizable interface with theme switching, sorting options, and a cool receipt generator. All data is stored locally on your device, making it lightweight and portable—no server needed! (I made it for my personal use trying to avoid my excessive spending habbits you can tweak according to your usage) 

## Features
- **Track Debts:** Log outgoing expenses with ease.
- **Receipt Generation:** Generate stylish receipts for each expense.
- **Sorting:** Sort expenses in ascending or descending order.
- **Themes:** Switch between multiple themes for a personalized look.
- **Local Storage:** Data persists on your device using browser's local storage.
- **Single HTML File:** All functionalities embedded in one simple file.

## Usage
1. Download or copy the `index.html` file.
2. Open it with any modern web browser (Chrome, Firefox, Edge, etc.).

6. Generate receipts for your records.
7. Change theme to suit your style.

## Limitations
- Data is stored locally; it won't sync across devices.
- Designed for personal use and small-scale tracking.
